# Embudo MPM

Embudo de innovación interactivo (HTML/CSS/JS puro, sin dependencias externas) para visualizar y dar seguimiento a las iniciativas de innovación de MPM a lo largo de 4 etapas:

1. Ideación
2. Incubación
3. Piloto
4. Implementación

## Uso

Abre `embudo_mpm.html` directamente en el navegador. No requiere instalación ni servidor.
